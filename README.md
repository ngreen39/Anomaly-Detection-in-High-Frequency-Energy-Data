# Anomaly Detection in Energy Data

A streamlined machine learning project to identify outliers and irregular consumption patterns in high-frequency household power data.

## Quick Start
1. Unzip `household_power_consumption.txt.zip`.
2. Run `final.ipynb` to view the analysis and model results.

##  Overview
* **Data:** 4 years of 1-minute interval energy measurements.
* **Process:** Data cleaning, cyclical feature engineering, and statistical anomaly detection.
* **Tech:** Python (Pandas, Scikit-Learn, Matplotlib).
